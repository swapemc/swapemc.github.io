ImguiLib

A faithful Dear ImGui-styled UI library for Roblox Luau. Dense, flat, monospace — built for scripts, exploits, and developer tools.

Luau Roblox Dear ImGui Style
01

Installation

Load the library at the top of your script using loadstring and an HTTP request to wherever you host the file.

Luau 
local ImGui = loadstring(game:HttpGet("https://your-host.com/ImGuiLib.lua"))()
ⓘ
The library attempts to parent its ScreenGui to CoreGui first (exploit environments), and falls back to PlayerGui automatically.
02

Quick Start

Create a window, add a tab, then add widgets to the tab. Everything chains off the objects returned.

Luau 
local ImGui = loadstring(game:HttpGet("https://your-host.com/ImGuiLib.lua"))()

-- 1. Create a window
local Window = ImGui.CreateWindow("My Tool")

-- 2. Add a tab
local Tab = Window:AddTab("Main")

-- 3. Add widgets to the tab
Tab:AddText("Hello, world!")

Tab:AddButton("Click me", function()
    print("Clicked!")
end)

Tab:AddToggle("God Mode", false, function(value)
    print("God mode:", value)
end)
03

CreateWindow

The entry point of the library. Returns a Window object.

Signature
ImGui.CreateWindow(title: string, options?: table) → Window
ParameterTypeDescription
titlestringText shown in the title bar.
options.Widthnumber?Window width in pixels. Default: 430
options.Heightnumber?Window height in pixels. Default: 480
options.PositionUDim2?Initial screen position. Default: centered.
options.ToggleKeyEnum.KeyCode?Key to show/hide the window. Default: RightShift
Example
local Window = ImGui.CreateWindow("My Tool", {
    Width     = 500,
    Height    = 600,
    Position  = UDim2.new(0.5, -250, 0.5, -300),
    ToggleKey = Enum.KeyCode.Insert,
})
04

Window API

Methods available on the Window object returned by CreateWindow.

method
Window:AddTab(name: string) → Tab

Creates a new tab in the menu bar and returns a Tab object you can add widgets to. The first tab added is auto-selected.

method
Window:SelectTab(name: string)

Programmatically switches to the tab with the given name.

method
Window:Toggle()

Toggles the window's visibility. Also triggered by the keybind.

method
Window:SetVisible(visible: boolean)

Sets visibility directly without toggling.

method
Window:SetToggleKey(key: Enum.KeyCode)

Changes the keybind used to show/hide the window at runtime.

Example
Window:SetToggleKey(Enum.KeyCode.Insert)
method
Window:SetTheme(overrides: table)

Applies colour overrides to the window. Only the keys you provide are changed — unspecified keys keep their defaults. See the Theming section for all available keys.

method
Window:Destroy()

Removes the ScreenGui and cleans up the window entirely.

05

Tabs

Tabs are created on the Window and act as the container for all widgets. You can have as many tabs as you need — they render as buttons in the menu bar.

Example
local MainTab     = Window:AddTab("Main")
local CombatTab   = Window:AddTab("Combat")
local SettingsTab = Window:AddTab("Settings")

-- Add widgets to each tab independently
MainTab:AddText("Main tab content")
CombatTab:AddText("Combat tab content")
06

AddText

Renders a static text label. Useful for section headings or descriptions.

Signature
Tab:AddText(text: string, disabled?: boolean) → { SetText: function }
Example
Tab:AddText("Hello, world!")
Tab:AddText("This is greyed out", true)  -- disabled/dim colour

local label = Tab:AddText("Score: 0")
label:SetText("Score: 100")  -- update later
07

AddButton

A clickable button. Width auto-fits to the label text, ImGui-style.

Signature
Tab:AddButton(text: string, callback?: function) → { SetText, SetCallback }
Example
Tab:AddButton("Teleport to Spawn", function()
    local spawn = workspace:FindFirstChild("SpawnLocation")
    if spawn then
        game.Players.LocalPlayer.Character.HumanoidRootPart.CFrame
            = spawn.CFrame
    end
end)
08

AddCheckbox

A boolean toggle rendered as a small square checkbox, identical to Dear ImGui's checkbox widget.

Signature
Tab:AddCheckbox(text: string, default: boolean, callback?: function)
    → { GetValue, SetValue, SetCallback }
Example
local godMode = Tab:AddCheckbox("God Mode", false, function(value)
    local char = game.Players.LocalPlayer.Character
    if char then
        char.Humanoid.MaxHealth = value and math.huge or 100
    end
end)

-- Read or set value later
print(godMode:GetValue())
godMode:SetValue(true)
09

AddRadioGroup

Renders a horizontal row of mutually exclusive radio buttons — only one option can be selected at a time.

Signature
Tab:AddRadioGroup(items: table, default: string, callback?: function)
    → { GetSelected, SetCallback }
Example
Tab:AddRadioGroup(
    {"radio a", "radio b", "radio c"},
    "radio a",
    function(selected)
        print("Selected:", selected)
    end
)
10

AddSliderInt / AddSliderFloat

A draggable slider. The current value is printed inside the bar, just like Dear ImGui. AddSliderInt snaps to whole numbers; AddSliderFloat gives 3 decimal places.

Signatures
Tab:AddSliderInt(text: string, min: number, max: number, default: number, callback?: function)
    → { GetValue, SetValue, SetCallback }

Tab:AddSliderFloat(text: string, min: number, max: number, default: number, callback?: function)
    → { GetValue, SetValue, SetCallback }
Example
local speed = Tab:AddSliderInt("Walk Speed", 0, 200, 16, function(v)
    game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = v
end)

local fov = Tab:AddSliderFloat("FOV Size", 0.0, 1.0, 0.5, function(v)
    print("FOV:", v)
end)

-- Set value programmatically
speed:SetValue(32)
print(fov:GetValue())
11

AddInputText

A single-line text input field. The callback fires on focus lost and optionally on every keystroke.

Signature
Tab:AddInputText(text: string, default: string, callback?: function)
    → { GetText, SetText, SetCallback }
Example
local nameBox = Tab:AddInputText("Target Player", "", function(text, enterPressed)
    if enterPressed then
        print("Searching for:", text)
    end
end)
✓
The callback receives two arguments: text (current string) and enterPressed (bool — true when the user hit Enter to confirm).
12

AddInputInt / AddInputFloat

Numeric inputs with − and + stepper buttons on either side, matching ImGui's InputInt / InputFloat widgets.

Signatures
Tab:AddInputInt(text: string, default: number, callback?: function, step?: number)
    → { GetValue, SetValue, SetCallback }

Tab:AddInputFloat(text: string, default: number, callback?: function, step?: number)
    → { GetValue, SetValue, SetCallback }
Example
-- Integer, stepping by 5
Tab:AddInputInt("Kill Aura Range", 20, function(v)
    print("Range:", v)
end, 5)

-- Float, stepping by 0.01
Tab:AddInputFloat("Gravity Scale", 1.0, function(v)
    workspace.Gravity = v * 196.2
end, 0.01)

AddCombo

A dropdown combo box. Shows the selected value with a small arrow button; clicking opens a floating popup list.

Signature
Tab:AddCombo(text: string, options: table, default: string, callback?: function)
    → { GetSelected, SetCallback }
Example
Tab:AddCombo("Aim Part", {"Head", "HumanoidRootPart", "Torso"}, "Head",
    function(selected)
        print("Aiming at:", selected)
    end
)
14

AddColorEdit

Displays a colour swatch and RGBA float values, matching ImGui's ColorEdit4 widget. Click the swatch to open a colour picker (if extended in your implementation).

Signature
Tab:AddColorEdit(text: string, default: Color3, callback?: function)
    → { GetColor, SetColor, SetCallback }
Example
local espColor = Tab:AddColorEdit("ESP Color",
    Color3.fromRGB(255, 0, 0),
    function(color)
        print(color.R, color.G, color.B)
    end
)

-- Set programmatically
espColor:SetColor(Color3.fromRGB(0, 255, 0))

AddListbox

A scrollable list of selectable items. The currently selected item is highlighted in ImGui's header blue.

Signature
Tab:AddListbox(text: string, items: table, default: string, callback?: function, visibleRows?: number)
    → { GetSelected, SetCallback }
Example
Tab:AddListbox("Players", {"Alice", "Bob", "Charlie", "Dave"},
    "Alice",
    function(selected)
        print("Selected:", selected)
    end,
    4  -- show 4 rows before scrolling
)
16

AddCollapsingHeader

A collapsible section header. Returns a Section object — you can call any widget method on it, and those widgets will appear indented inside the collapsed region.

Signature
Tab:AddCollapsingHeader(text: string, defaultOpen?: boolean) → Section
Example
local section = Tab:AddCollapsingHeader("Aim Settings", true)

-- All widget methods work on the returned Section
section:AddCheckbox("Silent Aim", false, function(v) print(v) end)
section:AddSliderInt("Smoothness", 0, 100, 50, function(v) print(v) end)
section:AddButton("Reset", function() print("Reset!") end)
ⓘ
The Section returned by AddCollapsingHeader supports every widget method: AddText, AddButton, AddCheckbox, AddSliderInt, AddSliderFloat, AddInputText, AddInputInt, AddInputFloat, AddCombo, AddColorEdit, AddListbox, AddSeparator, and even nested AddCollapsingHeader.
17

AddSeparator

A thin horizontal rule to visually separate groups of widgets.

Example
Tab:AddText("Group A")
Tab:AddButton("Action 1")
Tab:AddSeparator()   -- ─────────────────
Tab:AddText("Group B")
Tab:AddButton("Action 2")

Theming

Call Window:SetTheme({}) with a table of overrides. Any key you don't specify keeps its default Dear ImGui dark theme value.

Example
Window:SetTheme({
    TitleBgActive = Color3.fromRGB(80, 40, 120),  -- purple title bar
    SliderGrab    = Color3.fromRGB(200, 80, 80),   -- red slider fill
    CheckMark     = Color3.fromRGB(80, 200, 120),  -- green checkmarks
    Header        = Color3.fromRGB(80, 40, 120),  -- selected items
})

20

Keybind

The window can be shown/hidden with a keyboard shortcut. Default is RightShift. Set it in CreateWindow or change it later:

Example
-- At creation time
local Window = ImGui.CreateWindow("Tool", { ToggleKey = Enum.KeyCode.Insert })

-- Or at runtime
Window:SetToggleKey(Enum.KeyCode.F4)
⚠
The keybind only fires when GameProcessedEvent is false — it won't trigger while typing in a text box or using in-game chat.

Full Example

A complete script demonstrating all widget types across multiple tabs.

Luau
local ImGui = loadstring(game:HttpGet("https://your-host.com/ImGuiLib.lua"))()

local Window = ImGui.CreateWindow("My Tool", {
    Width     = 430,
    Height    = 500,
    ToggleKey = Enum.KeyCode.RightShift,
})

-- ── MAIN TAB ──────────────────────────────────
local Main = Window:AddTab("Main")

Main:AddText("Hello, world!")
Main:AddSeparator()

Main:AddCheckbox("God Mode", false, function(v)
    local hum = game.Players.LocalPlayer.Character.Humanoid
    hum.MaxHealth = v and math.huge or 100
    hum.Health    = hum.MaxHealth
end)

Main:AddSliderInt("Walk Speed", 0, 200, 16, function(v)
    game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = v
end)

Main:AddSliderFloat("Gravity", 0.0, 5.0, 1.0, function(v)
    workspace.Gravity = v * 196.2
end)

Main:AddInputText("Target", "", function(t) print(t) end)

Main:AddCombo("Aim Part", {"Head","Torso","HumanoidRootPart"}, "Head",
    function(v) print(v) end)

Main:AddButton("Rejoin", function()
    game:GetService("TeleportService"):Teleport(game.PlaceId)
end)

-- ── VISUALS TAB ───────────────────────────────
local Visual = Window:AddTab("Visuals")

local espSection = Visual:AddCollapsingHeader("ESP", true)
espSection:AddCheckbox("Boxes",    true,  function(v) print(v) end)
espSection:AddCheckbox("Name Tags", false, function(v) print(v) end)
espSection:AddColorEdit("ESP Color", Color3.fromRGB(255,0,0),
    function(c) print(c) end)

-- ── SETTINGS TAB ──────────────────────────────
local Settings = Window:AddTab("Settings")

Settings:AddText("Theme")
Settings:AddRadioGroup(
    {"Default", "Purple", "Red"}, "Default",
    function(v)
        local themes = {
            Default = Color3.fromRGB(41, 74, 122),
            Purple  = Color3.fromRGB(80, 40, 120),
            Red     = Color3.fromRGB(120, 30, 30),
        }
        Window:SetTheme({ TitleBgActive = themes[v] })
    end
)

Settings:AddSeparator()
Settings:AddButton("Destroy GUI", function()
    Window:Destroy()
end)
